## 요구 사항

- 간단한 매장 관리 시스템을 구현한다.
- 메뉴 그룹

    - [ ] 메뉴 그룹 목록을 조회한다.
- 메뉴
    - [ ] 메뉴를 생성한다.
    - [ ] 메뉴 가격을 수정한다.
    - [ ] 메뉴를 노출한다.
    - [ ] 메뉴를 숨긴다.
    - [ ] 메뉴 목록을 조회한다.
- 주문 테이블

    - [ ] 주문 테이블을 채운다.
    - [ ] 주문 테이블을 비운다.
    - [ ] 주문 테이블의 인원 수를 변경한다.
    - [ ] 주문 테이블 목록을 조회한다.
- 주문
    - [ ] 주문을 생성한다.
    - [ ] 주문을 수락한다.
    - [ ] 주문 배달 준비를 완료하였다.
    - [ ] 주문 배달을 시작한다.
    - [ ] 주문 배달을 완료처리한다.
    - [ ] 주문을 완료처리한다.
    - [ ] 주문 목록을 조회한다.
- 제품
    - [ ] 제품을 생성한다.
    - [ ] 제품 가격을 변경한다.
    - [ ] 제품 목록을 가져온다.
